# express-react-blueprint

## Get Started
Install dependencies.
```
yarn install
```

Boot up server.
```
yarn server
```

Run lint test.
```
yarn test:lint
```

Run unit tests.
```
yarn test:unit
```

Run unit tests with coverage report.
```
yarn test:coverage
```

View unit test coverage report.
```
yarn viewcoverage
```
